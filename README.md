# OGMA AI Assistant

Интеллектуальный Telegram-бот помощник, построенный на современных технологиях искусственного интеллекта. Проект объединяет возможности LLM (Large Language Models) с векторным поиском и персонализированной памятью.

## Функциональность

- **Интеграция с AI**: Поддержка OpenAI API и локальных моделей.
- **Векторный поиск**: Использование FAISS и Sentence Transformers для семантического поиска и долгосрочной памяти.
- **Система досье**: Автоматическое создание и обновление досье на пользователей и чаты.
- **Управление аккаунтами**: Привязка Telegram аккаунтов, сессии и авторизация.
- **Статистика**: Сбор и анализ действий пользователей.
- **Администрирование**: Управление ключами API, настройками моделей и пользователями.

## Технологический стек

- **Язык**: Python 3.10+
- **Telegram Framework**: Aiogram 3.x
- **База данных**: PostgreSQL (AsyncPG, SQLAlchemy 2.0)
- **Миграции**: Alembic
- **Кэширование**: Redis
- **ML & AI**: PyTorch, Sentence-Transformers, FAISS, OpenAI SDK
- **Веб-сервер**: FastAPI, Uvicorn

## Запуск

Для запуска бота выполните команду:

```bash
python main.py
```

## Структура проекта

- `tg_bot/` - Основная логика Telegram бота (хендлеры, клавиатуры, состояния)
- `database/` - Модели базы данных и логика подключения
- `neural_networks/` - Компоненты для работы с нейросетями и векторным поиском
- `config_data/` - Загрузка и валидация конфигурации
- `alembic/` - Миграции базы данных
